# ระบบลงทะเบียนเรียน
- ตัวอย่าง https://webreg.comsciproject.net
	- รหัสผ่านของตัวอย่างคือ : 1234
- นักเรียนต้อง Login ก่อนถึงจะเลือกวิชาเพื่อลงทะเบียนได้ (มีการป้องการไม่ใช้เข้าหน้า โดยไม่ได้ Login)
- Subdomain name ที่ใช้ตรวจการบ้านคือ :  http://reg.localhost
- เขียนโปรแกรม PHP ด้วย Concept MVC  ต่อจาก Lab ที่เรียน
	- กรณีไม่เป็น MVC (ตามที่สอน) จะปรับเปลี่ยนคะแนนตามสมควร
- ความสามารถของโปรแกรม
	- ผู้ใช้ Login และ Logout ได้
	- เลือกวิชาที่จะลงทะเบียนของนักเรียนคนนั้นๆได้
		- ต้องเป็นวิชาที่ยังไม่เคยลง จะลงทะเบียนซ้ำไม่ได้
		- สามารถถอนวิชาที่ลงทะเบียนไปแล้วได้
	- มีการ  confirm เมื่อจะลงทะเบียน หรือ ถอนวิชาได้            
	- มีการปรับแต่ UX/UI ที่เหมาะสม
